1. Regular Expression Guide:¶

  • [0-9] : Matches any digit
  • \d : Shortcut for [0-9]
  • [^0-9] : Matches any non-digit character
  • \D : Shortcut for [^0-9]
  • \s : Matches any whitespace character
  • \S : Matches any non-whitespace character
  • \w : Matches any alphanumeric character
  • \W : Matches any non-alphanumeric character

2. Example: Dot (.) in regex matches any single character except newlines.¶

Use \. or [.] to explicitly match a period.

# Matching patterns with `D.A`
# `D.A` matches a 'D' followed by any single character (except a newline), and then an 'A'.
pattern = re.compile(r'D.A')
print('DAA', pattern.search("DAA"))  
print('D1A', pattern.search("D1A")) 
print('d1A', pattern.search("d1A"))  
print('D  A', pattern.search("D  A")) 
print('d0A D1A 0111', pattern.search("d0A D1A 0111"))  

# Matching patterns with two characters between 'D' and 'A'
# `D..A` matches a 'D' followed by any two characters, and then an 'A'.
pattern = re.compile(r'D..A')
print('DAA', pattern.search("DAA"))  
print('D1A', pattern.search("D1A"))  
print('d1A', pattern.search("d1A"))  
print('D  A', pattern.search("D  A"))  
print('d0A D1A 0111', pattern.search("d0A D1A 0111"))  

# Matching a literal period with `D\.A`
# `D\.A` matches a 'D' followed by a literal period ('.') and then an 'A'.
pattern = re.compile(r'D\.A')
print('D\.A - ', pattern.search("D.A")) 
print('D\.A - ', pattern.search("DDA"))

pattern = re.compile(r'D\..A')
print('D\..A - ', pattern.search("D..A"))  
print('D\..A - ', pattern.search("D.AA")) 
print('D\..A - ', pattern.search("DDA"))

4. Example: Repetition using ?, *, +, and {m,n}.¶

# Pattern 'D?A': Matches 0 or 1 occurrence of 'D' followed by 'A'
pattern = re.compile(r'D?A')
print('D?A', pattern.search("A"))
print('D?A', pattern.search("DA"))
print('D?A', pattern.search("DDDDDDA"))

# Pattern 'AD{2}A': Matches 'A' followed by exactly 2 'D's and another 'A'
pattern = re.compile(r'AD{2}A')
print('AD{2}A', pattern.search("ADA"))
print('AD{2}A', pattern.search("ADDA"))

# Matching patterns with `AD{2,6}A`
# `AD{2,6}A` matches an 'A', followed by between 2 to 6 occurrences of 'D', and then another 'A'.

pattern = re.compile('AD{2,6}A')
print('AD{2,6}A - ', pattern.search("ADDA"))
print('AD{2,6}A - ', pattern.search("ADDDA"))
print('AD{2,6}A - ', pattern.search("ADDDDDDA"))
print('AD{2,6}A - ', pattern.search("ADDDDDDDDDA"))

6. Regular Expression Library Function Usage¶

Functions such as match, search, findall, and split are commonly used with regular expressions in Python.

  • match: Checks for a match only at the beginning of the string.
  • search: Searches the entire string for a match.
In [100]:
# Compile a regex pattern to match one or more lowercase letters.
pattern = re.compile('[a-z]+')

# `match` only checks the beginning of the string.
print('match Dave:', pattern.match('Dave'))  # Does not match because 'D' is uppercase.
print('match dave:', pattern.match('dave'))  # Matches because 'dave' starts with lowercase letters.

# `search` scans the entire string for a match.
print('search Dave:', pattern.search("Dave"))  # Matches because lowercase letters are found in the string.

match Dave: None
match dave: <re.Match object; span=(0, 4), match='dave'>
search Dave: <re.Match object; span=(1, 4), match='ave'>
  • findall: returns all matches of the regular expression in the string as a list.
In [101]:
# Compile a regex pattern to match one or more lowercase letters.
pattern = re.compile('[a-z]+')
print('Findall [a-z]+:', pattern.findall('Game of Life in Python'))  # Finds all sequences of lowercase letters.

# Compile a regex pattern to match one or more alphabetic characters (both uppercase and lowercase).
pattern2 = re.compile('[A-Za-z]+')
print('Findall [A-Za-z]+:', pattern2.findall('Game of Life in Python'))  # Finds all alphabetic sequences.

Findall [a-z]+: ['ame', 'of', 'ife', 'in', 'ython']
Findall [A-Za-z]+: ['Game', 'of', 'Life', 'in', 'Python']

Check existence of matches using findall
Use findall to check whether any matches exist for a given pattern.

pattern = re.compile('[a-z]+')
